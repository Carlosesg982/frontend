## git clone [URL_DE_ESTE_REPO]

## Ejecución Local

1. Instalar dependencias: `npm install`

2. Configurar el archivo `.env`
   NEXT_PUBLIC_URL_BACKEND

3. Iniciar el servidor: `npm run dev`.

4. Estructura del Proyecto

- `/app` Páginas y rutas de la aplicación.

- `/components` Componentes reutilizables de PrimeReact.

- `/store` Configuración de Redux Toolkit (Slices para vehículos y movimientos).

- `/lib` Configuración de Axios para consumo de la API.
